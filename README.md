# CI/CD Java Project

Проект демонстрирует настройку CI/CD с помощью GitHub Actions.

## Функции
- Простые математические методы (`add`, `factorial`, `square`)
- JUnit тесты
- Автоматическая сборка и тестирование при каждом коммите
- Генерация документации как пример CD шага


## CI запускается автоматически на ветках:
- main
- development
- release
